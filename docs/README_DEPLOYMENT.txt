🚀 PAȘI RAPIZI (3 comenzi):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1️⃣  INSTALEAZĂ DEPENDENȚELE:
      npm install
      (Ia 2-5 minute prima data...)

  2️⃣  CONSTRUIEȘTE PENTRU PRODUCȚIE:
      npm run build
      (Crează folder 'build/' cu ~200KB)

  3️⃣  ALEGE DEPLOYMENT:
      Vercel:     vercel deploy
      Netlify:    netlify deploy --prod --dir=build
      GitHub:     npm run deploy
      Local:      npm start
